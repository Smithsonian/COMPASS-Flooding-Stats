Dataset: Fluxes of carbon dioxide, methane, and nitrous oxide and associated environmental data from estuarine wetlands in the Pacific Northwest, USA

Authors: Christopher Janousek, Trevor Williams, Maggie McKeon, Scott Bridgham, Craig Cornu, Heida Diefenderfer, Amy Borde, Matt Norwood, Jude Apple, Laura Brophy

Any use of this dataset must include a citation. The DOI: 10.25573/serc.27161883

This data release contains a synthesis of chamber-based flux measurements. The data itself is housed in separate .csv files which can be joined by the chamber_id and/or site_id attributes. The files contain version one of the publicly available tidal wetland flux data. 

Janousek_et_al_2025_site_level.csv - Contains positional and descriptive information on site locations.
Janousek_et_al_2025_chamber_level.csv - Contains positional and descriptive information on chamber locations.
Janousek_et_al_2025_chamber_timeseries.csv - Contains raw time-series information for all chamber flux measurements.
Janousek_et_al_2025_porewater_level.csv - Contains associated porewater information at the site locations.
Janousek_et_al_2025_soil_temperature.csv - Contains associated soil temperature information at the site locations.
Janousek_et_al_2025_species.csv - Contains information on the dominant plant species present at site and/or chamber locations.

metadata.xml - Contains a full suite of descriptive metadata with all attributes and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual, and interactive version of metadata.xml for ease of use.
